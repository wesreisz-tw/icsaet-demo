# icsaet-demo

A minimal Python script project.

## Requirements

- Python 3.13 or higher

## Installation

```bash
pip install -e .
```

## Usage

Run the command:

```bash
icsaet-demo
```

Or run directly with Python:

```bash
python -m icsaet_demo.main
```

## Development

Install development dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```

Run linter:

```bash
ruff check .
```

Format code:

```bash
black .
```

Type checking:

```bash
mypy icsaet_demo
```
